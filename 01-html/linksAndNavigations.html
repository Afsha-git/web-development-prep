1. Basic Link Syntax

Standard Link:

Click here to visit Google

Syntax:

<a href="URL">Link Text</a>

Link Attributes:

  • href - URL destination (required)
  • target - Where to open the link
  • title - Tooltip text on hover
  • rel - Relationship between pages

Example with Title:

GitHub (hover to see tooltip)

2. External Links (Outside Your Website)

Open in Same Tab (default):

Wikipedia - Same Tab

<a href="https://www.wikipedia.org">Wikipedia</a>

Open in New Tab:

YouTube - New Tab

<a href="https://www.youtube.com" target="_blank">YouTube</a>

Open in New Tab (Secure - Recommended):

Twitter - Secure New Tab

<a href="URL" target="_blank" rel="noopener noreferrer">Link</a>

Target Attribute Values:

Value Description
_self Default - Same window/tab
_blank New window/tab
_parent Parent frame
_top Full body of window

3. Internal Links (Same Website)

Absolute Path:

My GitHub Repository

<a href="https://yoursite.com/page.html">Page</a>

Relative Path - Same Folder:

Day 1 Practice

<a href="page.html">Page in same folder</a>

Relative Path - Subfolder:

Portfolio Project

<a href="folder/page.html">Page in subfolder</a>

Relative Path - Parent Folder:

Parent Folder

<a href="../page.html">Page in parent folder</a>

Path Examples:

Path Meaning
page.html Same folder
folder/page.html Inside subfolder
../page.html One folder up
../../page.html Two folders up
/page.html Root directory

4. Anchor Links (Jump to Section)

Jump to Different Sections on THIS Page:

  • Jump to Basic Links
  • Jump to External Links
  • Jump to Internal Links
  • Jump to Top of Page

How It Works:

Step 1: Add ID to target section

<div id="section-name">Content</div>

Step 2: Link to it using #

<a href="#section-name">Jump to Section</a>

Jump to Section on Another Page:

Day 1 Topics Section

<a href="page.html#section-id">Link</a>

5. Email & Phone Links

Email Link (Opens Email Client):

Send me an email

<a href="mailto:email@example.com">Email Me</a>

Email with Subject & Body:

Email with Subject

<a href="mailto:email@example.com?subject=Hello&body=Message">Email</a>

Email with CC and BCC:

Email with CC & BCC

<a href="mailto:to@email.com?cc=cc@email.com&bcc=bcc@email.com">Email</a>

Phone Link (Click to Call):

Call: +91-9876543210

<a href="tel:+1234567890">Call Us</a>

SMS Link:

Send SMS

<a href="sms:+1234567890?body=Message">Send SMS</a>

6. Download Links

Download File:

Download Resume (PDF)

<a href="file.pdf" download>Download File</a>

Download with Custom Filename:

Download Resume (Custom Name)

<a href="file.pdf" download="custom-name.pdf">Download</a>

Different File Types:

  • PDF Document
  • Image File
  • ZIP Archive
  • JavaScript File
