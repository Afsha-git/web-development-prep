5. ID Attribute (Unique)

IDs should be unique - only ONE element can have a specific ID.

This paragraph has a unique ID "special-paragraph". Only THIS element can have this ID on the entire page.

Regular paragraph without ID.

IDs are useful for:

  • JavaScript targeting (getElementById)
  • CSS specific styling
  • Anchor links (jump to section)
  • Form label connections

6. CLASS vs ID - When to Use?

Feature CLASS ID
Usage Multiple elements ONE element only
Syntax class="name" id="name"
CSS Selector .classname #idname
JavaScript getElementsByClassName() getElementById()
Reusable? ✅ Yes ❌ No (unique)
Multiple values? ✅ Yes (space-separated) ❌ No (only one ID)
Best for Styling groups of elements Unique sections, JavaScript

9. What I Learned Today

  1. <div> is block-level, takes full width
  2. <span> is inline, takes only needed width
  3. class can be used on multiple elements
  4. id must be unique on the page
  5. Classes use .classname in CSS
  6. IDs use #idname in CSS
  7. Elements can have multiple classes
  8. Divs can be nested inside each other
